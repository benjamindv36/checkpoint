# Frontend Components

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend components.

## When to use this skill:

- When creating or modifying React, Vue, Svelte, or web components
- When defining component props, interfaces, or prop types
- When implementing component state management or lifecycle methods
- When building reusable UI elements like buttons, cards, modals, or forms
- When breaking down large components into smaller, focused components
- When implementing component composition patterns
- When lifting state up to parent components or using context
- When documenting component APIs and usage examples
- When working in component directories or .jsx, .tsx, .vue, .svelte files
