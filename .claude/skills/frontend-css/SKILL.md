# Frontend CSS

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend CSS.

## When to use this skill:

- When writing or modifying CSS, SCSS, or CSS modules
- When applying Tailwind utility classes or other CSS frameworks
- When using styled-components, CSS-in-JS, or emotion
- When defining or using design tokens (colors, spacing, typography)
- When styling UI components, layouts, or page elements
- When working with CSS methodologies like BEM or utility-first approaches
- When optimizing CSS for production with purging or tree-shaking
- When working in .css, .scss, .module.css files, or styling within components
