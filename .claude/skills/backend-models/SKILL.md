# Backend Models

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend models.

## When to use this skill:

- When creating or modifying ORM model files or entity definitions
- When defining database table structures and model attributes
- When implementing model relationships, foreign keys, or associations
- When adding database constraints (NOT NULL, UNIQUE, foreign keys)
- When implementing model validation rules at the model or database level
- When creating indexes on foreign keys or frequently queried fields
- When adding or updating timestamps (created_at, updated_at) on models
- When working in model directories or schema definition files
