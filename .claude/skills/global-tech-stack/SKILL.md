# Global Tech Stack

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle global tech stack.

## When to use this skill:

- When making technology or framework choices for implementation
- When adding new dependencies, libraries, or packages to the project
- When setting up integrations with third-party services
- When configuring build tools, linters, formatters, or testing frameworks
- When implementing features using the project's defined database, ORM, or caching layer
- When working with the specified frontend framework, CSS framework, or UI components
- When setting up authentication, email services, or monitoring tools
- Use this skill universally to ensure technology consistency across the project
