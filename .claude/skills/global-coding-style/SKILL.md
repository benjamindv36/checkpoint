# Global Coding Style

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle global coding style.

## When to use this skill:

- When writing any code in any programming language or framework
- When naming variables, functions, classes, files, or modules
- When creating or refactoring functions to be small and focused
- When applying automated formatting or linting to code
- When choosing descriptive, meaningful names for code elements
- When removing dead code, unused imports, or commented-out blocks
- When extracting common logic into reusable functions or modules
- When ensuring consistent indentation and code structure
- Use this skill universally across all coding tasks in the project
