# Global Error Handling

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle global error handling.

## When to use this skill:

- When implementing error handling or exception handling in any code
- When writing try-catch blocks, error boundaries, or error handlers
- When validating input and failing fast with clear error messages
- When handling errors from external services, APIs, or database calls
- When implementing retry strategies with exponential backoff
- When providing user-friendly error messages without exposing technical details
- When implementing graceful degradation for non-critical failures
- When cleaning up resources in finally blocks or equivalent mechanisms
- Use this skill universally whenever implementing error handling logic
