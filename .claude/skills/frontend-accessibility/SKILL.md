# Frontend Accessibility

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend accessibility.

## When to use this skill:

- When creating or modifying UI components, forms, buttons, or interactive elements
- When implementing keyboard navigation and visible focus indicators
- When adding semantic HTML elements (nav, main, button, article, etc.)
- When implementing ARIA attributes for complex components
- When adding alt text for images or labels for form inputs
- When ensuring proper color contrast ratios in designs
- When managing focus in dynamic content, modals, or single-page applications
- When creating accessible navigation menus, dropdowns, or accordions
- When working with heading structures and document outlines
- When testing with screen readers or accessibility testing tools
