# Frontend Responsive

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive.

## When to use this skill:

- When implementing responsive layouts or media queries
- When creating mobile-first designs that scale up to larger screens
- When working with CSS breakpoints (mobile, tablet, desktop)
- When using flexbox, grid, or fluid containers for responsive layouts
- When implementing touch-friendly UI elements with appropriate tap target sizes
- When using relative units (rem, em, %, vw, vh) instead of fixed pixels
- When optimizing images and assets for different screen sizes
- When testing UI across multiple device sizes and orientations
- When working on responsive navigation menus or adaptive components
